# Desktop Benchmark Starter

QitOS v0.5 现在有了一条明确的官方多模态主线：

- `DesktopEnv`
- `desktop-starter` benchmark
- `qitos.recipes.desktop.osworld_starter`
- `examples/real/openai_cua_agent.py`
- qita visual replay

如果你要做 computer-use 研究，这是当前最值得采用的 canonical path。

## 为什么这是 v0.5 的发布路径

因为它第一次把以下东西完整地接在了一起：

- benchmark family
- canonical baseline agent
- 统一 runner output
- trace（运行记录）artifacts
- qita visual debugging
- docs/tutorial story

如果你需要真实 benchmark adapter，而不是 starter path，请转向 `osworld`。
